# Colaboração e revisão

## Quando resolver sozinha com o Claude Code

- Dúvidas de conceito (o que é uma medida, por que separar em dimensões)
- Erros de sintaxe DAX
- Ajuste de tratamento/limpeza de uma base nova

## Quando trazer para revisão conjunta

- Definição do que vai para o dashboard final (quais perguntas de negócio importam)
- Números que não batem com o que você já sabia da área
- Decisões sobre acesso/permissão do dashboard publicado

## Como registrar uma dúvida para revisão

Ao encontrar algo que não resolve sozinha, anote:

1. Em qual etapa (`01` a `05`) surgiu
2. O que você tentou
3. O que esperava vs. o que aconteceu

Isso torna a revisão conjunta rápida e objetiva, em vez de refazer o raciocínio do zero.
