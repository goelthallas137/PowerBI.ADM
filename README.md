# Mentoria Power BI — roteiro didático

Material para aprender, sozinha e no seu ritmo, o fluxo completo de análise de dados usado para construir dashboards em Power BI — da entrada dos dados brutos até o dashboard publicado.

## Como usar

1. Abra esta pasta no **Claude Code** (ele lê `CLAUDE.md` automaticamente e já entende o roteiro).
2. Peça para começar pelo `01-fundamentos/`, ou vá direto à etapa em que estiver: `02-fluxo-de-trabalho/1` a `5`.
3. Sempre que tiver uma dúvida real (um dado que não bate, uma planilha bagunçada), leve o arquivo ou print junto — peça para aplicar o conceito da etapa ao seu caso.
4. Dúvidas que não se resolvem sozinha vão para `03-trabalho-conjunto/colaboracao-e-revisao.md`.

## Estrutura

```
CLAUDE.md                           → contexto que o Claude Code lê automaticamente
01-fundamentos/
  skills-e-ferramentas.md           → o que cada ferramenta faz e por quê
02-fluxo-de-trabalho/
  1-entrada-de-dados.md
  2-tratamento-e-limpeza.md
  3-modelagem-estrela.md
  4-medidas-dax.md
  5-dashboard-e-publicacao.md
03-trabalho-conjunto/
  colaboracao-e-revisao.md          → como pedir ajuda e revisar junto
templates/
  dax-measures.md                   → medidas DAX prontas para adaptar
```

Este roteiro é intencionalmente genérico: os exemplos usam dados administrativos/RH típicos (folha, headcount, absenteísmo), mas o método vale para qualquer base.
