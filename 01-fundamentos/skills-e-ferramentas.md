# Skills e ferramentas

Ferramentas de apoio que o Claude usa ao longo do fluxo. Você não precisa configurar nada — só saber o que cada uma faz para entender o que está acontecendo em cada etapa.

| Ferramenta | Para que serve | Etapa onde aparece |
|---|---|---|
| Leitura/edição de planilhas (Excel/CSV) | Abrir, limpar e reorganizar a base bruta | `02/1`, `02/2` |
| Perfilamento de dados | Checar nulos, duplicados, tipos e qualidade antes de modelar | `02/2` |
| Modelagem/visualização de dados | Montar o esquema estrela e escolher os gráficos certos | `02/3`, `02/5` |
| Fluxo analítico completo | Conduzir o processo ponta a ponta (coleta → dashboard) quando o pedido é amplo | todas |

## Conectores úteis (fontes externas)

| Conector | Uso |
|---|---|
| Google Drive | Importar planilhas direto de onde a base já é mantida |
| Notion | Registrar decisões e dúvidas do projeto |
| Microsoft Learn | Consultar documentação oficial de Power BI/DAX quando precisar aprofundar |

Nenhum desses é obrigatório — use o que já fizer parte da sua rotina de trabalho.
